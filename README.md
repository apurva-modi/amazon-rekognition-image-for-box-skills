## Amazon Rekognition Image For Box Skills

Sample application using Amazon Rekognition Image, Amazon S3, and AWS Lambda using Box Skills events

### To install:
git clone <repo URL>

git submodule update -init --recursive

npm install

See more detailed instructions and required additional resources here: https://aws.amazon.com/blogs/apn/applying-computer-vision-to-images-with-amazon-rekognition-aws-lambda-and-box-skills/
